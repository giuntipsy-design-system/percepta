Getting started

Overview

Percepta is the technical design system for Giunti Psychometrics. This repository documents how design decisions are implemented in code.

What it is

Percepta is the technical layer that turns design decisions into tokens, shared components, and documentation. It keeps UI implementation consistent across products.

What it solves

It reduces drift between design and code, standardizes UI behaviors, and provides a single source of truth for frontend implementation.

Who it is for

Frontend engineers building with Angular and PrimeNG.

Platforms

Angular + PrimeNG.

Design reference

Design foundations and brand guidelines live at design.giuntipsy.com.
