Select

Dropdown choice for a short list of mutually exclusive options.

Overview Grouping Usage Accessibility Properties

Selects are ideal when the user must choose one value from a limited list. Select relies on PrimeNG’s native behavior and accessibility while Percepta customizes appearance through tokens, not interaction.

Preview

Grouping (optgroups)

Use optgroups to organize long lists into meaningful clusters so users can scan faster.

Usage

Selects should be concise, labeled, and placed near the content they affect. Prefer a short list; if the list is long or searchable, consider autocomplete instead.

Guidelines

  • Use a clear label that matches the content users expect.
  • Keep option labels short and parallel in tone.
  • Default to a placeholder when no choice is selected.
  • Group long lists with optgroups instead of adding more variants.

Do

Provide context with a label and supporting helper text.

Applies to all team members.

Don’t

Rely on placeholder text without a clear label.

Accessibility

PrimeNG handles keyboard and ARIA behaviors. Keep visible labels and descriptive helper text to preserve accessible navigation.

  • Keyboard: Tab to focus, Space/Enter to open, arrows to move, Enter to select, Esc to close.
  • Always provide a visible label; use aria-label only when the label is hidden.
  • Announce helper or error text when present.
